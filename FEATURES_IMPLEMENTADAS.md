# AI Team App — Funcionalidades Implementadas

Use esta lista para evitar ideias duplicadas.

## Core
- Home com lista de projetos
- Página de agentes (/agents)
- Board Kanban com 8 colunas (Ideias → Backlog → Anna → Frank → Rask → Bruce → Ali → Done)
- CRUD de tasks com atribuição de agente
- Toggle “Forçar Opus” (badge no card)
- Runner v2 com seleção inteligente de modelo
- Progress log incremental (JSONB)
- Criação automática de PR (Bruce)

## Automação
- Crons de ideias (manhã 07:00 / noite 22:00) por projeto com `auto_ideas=true`
- Integração com Supabase DEV (dev_projects, dev_tasks, dev_task_comments)
